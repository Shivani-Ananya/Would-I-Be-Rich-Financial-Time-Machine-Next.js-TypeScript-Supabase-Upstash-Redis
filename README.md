# Would-I-Be-Rich-Financial-Time-Machine-Next.js-TypeScript-Supabase-Upstash-Redis
Would I Be Rich? (WIBR)  Would I Be Rich? (WIBR) is a full-stack financial simulation platform that enables users to evaluate the long-term impact of everyday spending decisions by comparing them against historical investment performance.
The application transforms routine expenses into interactive investment scenarios, helping users understand the concept of opportunity cost through data-driven visualizations.

Users can enter spending habits across categories such as food, shopping, subscriptions, transportation, and entertainment, then compare how those expenditures could have performed if invested in assets including cryptocurrencies, equities, index funds, precious metals, mutual funds, government-backed schemes, fixed-income instruments, and real estate.

#Key Features
Historical investment simulations using market data
Opportunity cost analysis through Real You vs Alternate You comparisons
Interactive financial dashboards and performance charts
Multi-category spending and investment comparison
USD and INR currency support with real-time exchange rate conversion
Responsive, modern user interface with an intuitive user experience
Serverless architecture for scalable and efficient processing
Fast data retrieval using intelligent caching mechanisms

#Technology Stack

Frontend: Next.js 15, React, TypeScript, Tailwind CSS
Backend: Next.js Serverless API Routes
Database: Supabase PostgreSQL
Caching: Upstash Redis
Market Data: Yahoo Finance APIs
Currency Conversion: Frankfurter API
Data Visualization: Recharts
Deployment: Netlify

#Project Objective

The objective of WIBR is to promote financial awareness by demonstrating how consistent spending patterns influence long-term wealth accumulation. By combining historical financial data, interactive simulations, and modern web technologies, the platform provides an educational environment for exploring investment outcomes and understanding the financial impact of everyday decisions.

#Disclaimer: The simulations and projections presented are based on historical market data and predefined assumptions. They are intended solely for educational and informational purposes and should not be interpreted as financial or investment advice.
